MapleLCMDistChronoBTv3 from SAT Race '19 with some new heuristics implemented.

# Compiling in Linux
To build and install, issue:
```
# getting the code
git clone https://github.com/arijitsh/mldcsat
cd mldcsat/simp
make
```
## Running
```
./glucose filename.cnf
```
